# Note: Model Identifiers (provider:model)

**Doc ID:** gen-dsl/note/model-identifiers

## Recommendation
Use a readable string form:
- `ollama:llama3:70b`
- `mlx:Qwen3.5-27B-4bit`
- `openai:gpt-5`

If no provider prefix is given, use `default_provider`.
